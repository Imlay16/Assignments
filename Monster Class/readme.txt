
    - Monster Class
        - 생성자에 받는 정보
            - 이름
            - 생명력
            - 공격력
            - 방어력
            - 속성 (불, 물, 바람, 땅)
                - 불은 바람에 강하나 물과 땅에는 약함
                - 물은 불에 강하나 바람에 약함
                - 땅은 불에 강하고 바람에 강함
                - 바람은 땅과 물에 강하나 불에는 약함

        - 자기 자신에게 데미지주는 함수
            - Ex) Hp 50
                - 함수(40)
                    - Hp 10
        - 다른 몬스터를 공격하는 함수
            - 인자로는 공격할 몬스터를 받습니다.
        - 데미지 연산
            - 데미지 최소 값은 1 입니다.
            - 데미지 연산은 다음과 같습니다.
                - 때리는애 공격력 - 맞는애 방어력
                - 속성이 상성이라면 데미지를 1.5배, 역상성이라면 0.5배, 무상성이라면 1배
                - 소수점은 무시
